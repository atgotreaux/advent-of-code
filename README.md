My solutions for [**Advent of Code**](https://adventofcode.com/), an Advent calendar of programming puzzles.

This project is powered by [**Spring Boot**](https://spring.io/projects/spring-boot/) and [**Spring Shell**](https://spring.io/projects/spring-shell/) to achieve a simple CLI application for solving AoC puzzles.

## Requirements

* [**Maven**](https://maven.apache.org/download.cgi)
* [**Java 21**](https://jdk.java.net/21/)
